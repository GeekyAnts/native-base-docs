Featured Apps

Having tried the free version NativeBase, Native Starter Kit and appreciated our product, Get on the mobile fast track with the featured apps of NativeBase, to build high-quality iOS and Android mobile apps.

Based on React Native and NativeBase, Native Starter Pro, Ecommerce Starter, Flat App Theme are beautifully designed with fusion of Redux and ES6, responsive iOS and Android app source code to quickly get started on your next project.

Check out some of the cool Apps and themes we have already built here.

1. Native Starter Pro

A 3 page theme: Login, SignUp and Dashboard, with Integrated Backend APIs.
Start your mobile project with ready-to-use User Auth and Dashboard with backend integration (Express, Node, MongoDB & JWT).


Learn more about Native Starter Pro With Backend 6.0.0

2. Dating App

The ideal starter kit / app script with all the needed UI elements alongwith Redux, NativeBase, CodePush and backend integration (Express, Node, MongoDB and Cloudinary) to build your iOS and Android dating app.


Learn more about React Native Dating App With Backend 5.2.0

3. E-Commerce Starter

A well structured React Native starter project to build your iOS and Android e-commerce app.
Fully customizable pages along-with a rich collection of UI elements specifically for an ideal e-commerce app.

Learn more about React Native E-Commerce Starter 6.0.0

4. Flat App Theme

A fascinating React Native starter kit with flat UI design, Redux and NativeBase components for your iOS and Android application.


Learn more about React Native Flat App Theme 6.0.0

5. Food Ordering App

The ideal starter kit / app script with all the needed UI elements alongwith Redux, NativeBase and CodePush to build your iOS and Android food ordering app like Seamless / Grubhub / Doordash / UberEats / Postmates / Delivery.com / Yelp Eat 24 / Foodler / Swiggy / FoodPanda / Zomato.


Learn more about React Native Food Ordering App 6.0.0

6. Messaging App with Backend

The ideal starter kit / app script with all the needed UI elements alongwith Mobx, NativeBase, CodePush and backend integration (MongoDB, Feathersjs) to build your iOS and Android messaging app.


Learn more about React Native Messaging App Theme with Backend 2.0.0-beta

7. Taxi App with Backend

The ideal starter kit with all the needed UI elements and Backend Integration to build your iOS and Android Taxi app like Uber / Ola / Lyft.

8. Social App Theme

The ideal starter kit / app script with all the needed UI elements alongwith Redux, NativeBase and CodePush to build your iOS and Android social app.

Learn more about React Native Social App Theme 6.0.0.

The ideal starter kit / app script with all the needed UI elements alongwith MobX and NativeBase to build your iOS and Android e-commerce app like Mona / JackThreads / Canopy / Flipp.
